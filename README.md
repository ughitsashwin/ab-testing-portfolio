# Ad Hoc Analysis & A/B Testing Portfolio

A personal project demonstrating skills in exploratory data analysis,
ad hoc investigation, and statistical A/B testing.

## Project structure
- `notebooks/` — Jupyter notebooks for each analysis
- `data/` — datasets used (see Data section below)
- `reports/` — exported charts and write-ups
- `src/` — reusable helper functions

## Tech stack
Python · pandas · scipy · statsmodels · seaborn
